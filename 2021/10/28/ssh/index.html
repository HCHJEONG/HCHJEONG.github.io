Github and SSH key setting

Posted on 2021-10-28 Edited on 2021-11-07

SSH?

  • Public / Private Key Pair using encryption technology like SHA256 widely accepted since 1970's

  • For example, in case of Github, you can register your one public key at ACCOUNT SETTING, which means this works in relation of your github account (not project-wise) to your specific computer and you should register at every computer respectively

Key Generation

  • at `/home/user/.ssh/ using linux utility

  • `ssh-keygen -t rsa -b 4096 -C "hcjeong@kakao.com"

  • Enter passphrase

  • This process will produce two files which contains public key and private key respectively

  • Moreover, this will produce key fingerprint and key randomart image

  • You need to check Agent id by using eval "$(ssh-agent -s)"

  • ssh-add ~/.ssh/id_rsa: This will complete the process

  • You can copy public key by clip < ~/.ssh/id_rsa.pub
